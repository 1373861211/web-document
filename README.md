## 前端历程

### 起源

还是觉得很有必要说一下写这个文档的初衷

其实很简单

参加了一些面试，发现很多前端基础性的东西掌握程度并不是那么透彻，所以想把它们都记录下来，留给自己温习也分享给有需要的小伙伴。在这段历程中，很多基础知识得到了巩固，更让我感到充实和满足的是分享所带来的那份成就感…

不知道能写多久，尽我所能努力打造自己的学习体系，一直在路上…
